Erklärung der Oberfläche

Das Menü des 4-Gewinnt-Spiel besteht aus 3 Tabs: "Game", "Player", "Options".

GAME-TAB:

Im Game-Tab kann ausgewält werden mit welchen Spielern gespielt werden soll, bzw. ob gegen eine 
Zufalls-KI gespielt werden soll.
Dabei darf nicht gegen sich selbst und nicht gegen einen Spieler mit der selben Farbe gespielt werden.

PLAYER-TAB
Hier können Spielerprofile angelegt, geändert und verwaltet werden. 
Eine Highscoreliste ist über den Knopf Highscore verfügbar.
Der Score berechnet sich folgendermaßen:
Jeder Sieg bringt 3 Punkte, jede Niederlage zieht einen ab. Ein Unentschieden ändert nichts.

Spielererstellung:
Soll ein neuer Spieler erstellt werden, so ist dabei zu beachten dass der Name des Spielers 
einzigartig ist und, falls er ein Bild
erhalten soll, dass der Dateiname des Bildes ebenfalss einzigartig ist.
Ein Bild ist nicht zwingend, es kann alternativ aus 3 Farben eine ausgewählt werden.
Das Bild, bzw. die Farbe füllen im GameWindow deine Felder.

OPTIONS-TAB:

Hier kann die Lautstärke der Musik eingestellt sowie das ReadMe eingesehen werden
